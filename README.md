# Hybrid_Imputation_and_Staged_Multilabel_Learning_for_Sleep_Apnea_Treatment_Recommendation
Master’s project (Anglia Ruskin University) applying machine learning to predict staged sleep apnea treatment recommendations using clinical data.

This repository contains my MSc Applications of Machine Learning module assignment at Anglia Ruskin University, Cambridge, UK. The project focuses on applying machine learning techniques to predict treatment pathways for obstructive sleep apnea (OSA) using real-world clinical data.

The work explores a full pipeline including data preprocessing, hybrid missing-value imputation, transformation of semi-structured treatment text into structured multilabel targets, and stage-wise supervised learning. Multiple models such as Logistic Regression, Random Forest, Linear SVM, and ensemble methods are evaluated.

The results demonstrate that early-stage treatment recommendations are more predictable than later stages, highlighting both the potential and challenges of applying machine learning in clinical decision support systems.
